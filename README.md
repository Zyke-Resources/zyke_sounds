## Disclaimer
By itself, this resource brings no functionality. It is a tool that can be used within other resources to enhance immersion.

This resource was developed to enhance [zyke_smoking](https://zykeresources.com/package/smoking), since no other available alternatives had the functionality I was looking for, and had massive delays when playing quick one-off sounds. This later expanded to [zyke_consumables](https://zykeresources.com/package/consumables), and it is now something I use throughout our newer releases frequently.

If you want to see how this resource can be integrated, check out [our documentation](https://docs.zykeresources.com/free-resources/sounds/exports-and-events).
